Boo, your new book companion

Project done for the Web Devepment Course, FII, Year 2

By Lau Mihaela Daiana and Blindu Andrada Mihaela

Introduction

Purpose and Product Scope

The purpose of the app is to act like a user friendly book annotation app and an online library. With user groups, friends , and a visual news section. The user can see when someone leaves a comment on a book, or wants to read somethig,etc. The inspiration of this app is the well known "Good reads".

Document Conventions

Within the whole web application, we only used a colour pallete of 5 colours (#3d284f, #57356A , #714E7D, #B790C2, #E3CEDD) and white(#fff). For fonts, we used -Nunito Sans

Intended Audience and Reading Suggestions

Our app only has one type of user intended, and that is a reader. They can join groups, make friends, add books to their library, leave comments and so on.

Reference

Our main reference is Good reads, a well known book annotating app

Overall Description

BOO is a new stand alone app meant to ask as a companion for the reader, as well as allow people to create a comunity of readers who love the same books.

Product Functions

    The App Will contain the following functions

  • Register/LogIn/LogOut/Change Password
  • Join Groups
  • Search Books
  • Leave comments/Other Anotations

Operating Environment

Our intention is to make the web application easy to use on any device, with it being platfom independent, we want it to be resposive.

User Documentation

The user can find the page "About" on the header of the aplication. Within it they can find a link to a "Help" Section meant to help them learn how to use the application

Dependencies

None

System Features

Join Groups

Within the app, the user can see the available groups alongside a description of it and can Join it if they wish to

Create Groups

The user is also capable of creating groups on their own

Search Books

The user can search for certain books and choose the genre too

Create Shelves

The user can create shelves and add books to any of the existent one (want to read, reading, already read)

Leave anotations on books

The user can leave comments and annotations on books which will be seen on the RSS news flux

App Structure

FrontEnd

Languages used

In it's current state, the web application uses a mix of HTML, SCSS, BEM and JavaScrip to make it as user friendly as possible. The app consists of a number of pages including a main feed, a group page, a search page and a book and library page.

User Interface

The user interface is simple but friendly and currenly contains 13 diferent pages.

BackEnd

Currently, the app is not functional, as we only worked to get the front end part done. We plan to use a mix of java scrip and php for the functionality down the road
